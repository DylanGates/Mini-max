- `AppDelegate.swift` — wired to create and own StatusBarController

<task type="auto">
  <name>Task 1: Create StatusBarController with 5 live indicators</name>
  <files>Mini-max/StatusBarController.swift</files>
  <action>
    Create @MainActor @Observable class StatusBarController.
    Own a single NSStatusItem (variable length).
    Use a custom NSView subclass as the status item's view — draw 5 indicator segments side by side.

    Each indicator segment:
    - Colored filled circle (8pt diameter): red/orange/purple/blue/green
    - Label text to the right of dot (hidden in ultra-compact mode)
    - Segments separated by 6pt gap

    Layout modes (enum StatusBarLayout: String, stored in UserDefaults key "minimax.statusbar.layout"):
    - .primary (default): dot + number, e.g. "● 18:42  ● 12  ● 5  ● 3  ● 2"
    - .ultraCompact: dot only "● ● ● ● ●", hover shows tooltip
    - .badge: dot + labeled number, e.g. "● 18:42  ● 12d  ● 5d  ● 3 events  ● 2 tasks"

    Data sources (read from shared singletons, observe with @Observable or NotificationCenter):
    - Pomodoro: PomodoroManager.shared.phase → format remaining as MM:SS; color red #FF453A
    - GitHub streak: compute from GitHubContributionStore.shared.contributionsByUser (consecutive days); color orange #FF9F0A
    - Learning streak: LearningStore.shared — count consecutive days with completedTopics; color purple #BF5AF2
    - Events today: CalendarManager.shared.events.count; color blue #0A84FF
    - Tasks pending: TaskStore.shared.pending.count; color green #30D158

    For live Pomodoro ticking: use a Timer (1s interval) on the main run loop, fired only when phase != .idle.
    Timer invalidates itself when phase returns to .idle.

    Use NSAttributedString to render the combined bar as a single string in the status item view, or draw segments with NSBezierPath + NSString.

    Avoid: multiple NSStatusItems (one is correct); system menu bar icons that conflict with existing StatusBarController.swift if one exists.
  </action>
  <verify>
    Build succeeds. App launches and shows dot indicators in menu bar.
    With a Pomodoro running, the timer label ticks every second.
  </verify>
